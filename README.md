# 🍽 Ordering System

## Vue2 + uni-app 的堂食 / 外卖一体化点单系统


## 📌 项目简介

本项目是一个基于 **Vue2（Options API）+ uni-app + uView UI** 开发的多端餐饮点单系统。

支持功能：

* 🏠 门店堂食
* 🛵 外卖配送
* 🎟 优惠券系统
* ⭐ 积分系统
* 📦 订单状态流转
* 🛠 管理员后台功能

---

## 🧱 技术栈

| 技术          | 说明             |
| ----------- | -------------- |
| Vue2        | Options API 架构 |
| uni-app     | 跨端开发框架         |
| uView UI    | UI 组件库         |
| uni.request | 网络请求封装         |
| 小程序 / H5    | 多端运行           |

---

# 📁 项目结构

```
ordering-system/
├─ common/
│  ├─ api.js          # 接口封装
│  ├─ request.js      # 网络请求封装
│  └─ config.js       # 全局配置
├─ pages/
│  ├─ index/          # 首页
│  ├─ menu/           # 菜单页
│  ├─ order/          # 订单列表
│  ├─ my/             # 我的
│  ├─ profile/        # 个人资料
│  ├─ coupon/         # 优惠券
├─ subPack/
│  └─ index/
│     ├─ indexSettlement.vue    # 支付详情
│     └─ indexPaysuccess.vue    # 支付成功
└─ unpackage/
```

# 👤 用户端功能

---

## 🏠 首页

* Banner 轮播
* 堂食 / 外卖入口
* 公告弹窗
* 积分展示
* 优惠券入口

---

## 🍜 菜单点餐

* 分类 + 菜品瀑布流
* 加减数量
* 购物车自动合计
* 自动匹配最优优惠券
* 堂食 / 外卖模式切换

## 💳 结算页

* 展示订单明细
* 显示配送地址
* 选择配送时间（u-picker）
* 选择 / 取消优惠券
* 模拟微信支付

* 清空购物车
* 返回支付成功页

## 📦 订单列表

支持功能：

* 管理员状态更新

* 昵称

* 配送地址（外卖必填）

## 🎟 优惠券系统

* 优惠券兑换

# 🛠 管理端功能

通过 `userType` 区分普通用户与管理员。

## 🍽 菜单管理

* 分类新增 / 编辑 / 删除

## 📦 订单管理

### 堂食

# 🔄 核心业务流程

## 🏠 堂食流程

浏览菜单 → 加入购物车 → 填写电话 → 支付 → 出餐

---

## 🛵 外卖流程

浏览菜单 → 加入购物车 → 校验地址 → 选择配送时间 → 支付 → 配送 → 送达
